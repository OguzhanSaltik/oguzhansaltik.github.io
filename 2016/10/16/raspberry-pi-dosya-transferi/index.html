Raspberry Pi Dosya Transferi

16 October 2016 on Raspberry Pi, SSH, Terminal, Dosya Transferi

Raspberry Pi içerisine SSH ile erişim yapacak olan bilgisayar üzerinden dosyanın nasıl yollanacağını içermektedir.

SSH Bağlantısı

Raspberry Pi ‘ye SSH bağlatısını sağlamak için ilgili blog yazımıza göz atabilirsiniz.

Terminali açıp scp FROM_ADDRESS USER_NAME@IP_ADDRESS:TO_ADDRESS formatında bir komut ile dosya transferi gerçekleştirmeye çalışacağız.

FROM_ADDRESS: SSH 'ın açıldığı bilgisayar, yani laptop üzerinden Raspberry Pi 'ye dosya yollanacaksa laptop üzerindeki dosyanın konumu.

USER_NAME: Raspberry Pi üzerinde dosyanın yollanacağı kullanıcının hesap adı.

IP_ADDRESS: Raspberry Pi 'ye ait IP adresi.

TO_ADDRESS: Raspberry Pi üzerinde dosyanın kaydedileceği dizinin konumu.

Şifremizi girmemizi istiyor, başarılı bir şekilde doğrulandıktan sonra aktarım tamamlanmış oluyor. Bu şekilde PC 'deki bir dosyayı Raspberry Pi içerisine SSH bağlantısı ile kolaylıkla kopyalayabilsiniz.
